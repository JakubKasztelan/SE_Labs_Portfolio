# Software Engineering Labs Portfolio

This repository contains a collection of software engineering tasks developed by Jakub Kasztelan at the Lodz University of Technology.

## Repository Structure

* **Labs/**: Contains UML diagrams created during laboratory sessions.
* **Project/**: Documentation and diagrams for the "Home Network Security Health-Check Tool".
* **SRS/**: Evolution of the Software Requirements Specification (SRS) documents following the IEEE 830 standard.

## Main Project Overview

The primary project in this repository is a tool designed to enable non-technical users to audit their local network security. Key features include:
* **Device Discovery**: Identifying active devices on the local network.
* **Vulnerability Audit**: Scanning for high-risk open ports and vulnerable services.
* **Security Scoring**: Translating technical weaknesses into a user-friendly 0-100 score.
* **Technical Reporting**: Generating PDF summaries for professional review.

Detailed project specification can be found in the **SRS/** folder.
