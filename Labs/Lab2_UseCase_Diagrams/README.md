# Lab 2: Use Case Diagrams

This laboratory exercise focuses on identifying boundaries, actors and functional requirements of example systems through Use Case Diagrams.

## Included Diagrams
* **ATM Use Case Diagram**: Models interactions between Bank Customers and Administrators with the ATM system.
* **Car Insurance Use Case Diagram**: Models the relationship between Clients and Insurance Agents.
* **Medical Clinic Use Case Diagram**: Models the workflows for Patients, Registration Staff and Doctors.
