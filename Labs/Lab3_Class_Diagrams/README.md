# Lab 3: Class Diagrams

This laboratory exercise focuses on the structural modeling of example systems using UML Class Diagrams to define objects, attributes and relationships.


## Included Diagrams
* **ATM Class Diagram**: Models the structural connections between entities such as `Account`, `Card`, `Transaction` and the `Bank` database.
* **Car Insurance Class Diagram**: Models the the connections between `Policy` and `Vehicle` and the attributes of `InsuranceAgent` and `Customer`.
* **Clinic Class Diagram**: Models the connections for entites such as `Patient`, `Staff`, `Appointment` and `MedicalRecord`.
