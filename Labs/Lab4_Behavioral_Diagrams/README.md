# Lab 4: Behavioral Diagrams

This laboratory exercise models the example systems behavior using Sequence, Activity and State diagrams.

## Included Diagrams
* **ATM Sequence Diagram**: A step-by-step visualization of the interaction between the User, ATM Interface and Bank Server during a withdrawal process.
* **Car Insurance Activity Diagram**: A flowchart representing the business logic involved in reporting and settling an insurance claim.
* **Clinic State Diagram**: Maps the statuses of a clinic appointment or visit (e.g., Scheduled, In Progress, Completed, Cancelled).
